# PartNApp

PartNApp aide des personnes à trouver des partenaires pour lancer des projets entrepreneuriaux.

L’objectif est de faciliter :

- la publication d’une demande de partenariat ;
- la recherche de personnes avec des intérêts ou projets similaires ;
- la mise en relation entre porteurs de projets ;
- le passage d’une idée à un projet réellement démarré.

Le produit est prévu sur mobile, iOS et Android, avec une version web séparée.
